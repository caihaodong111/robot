## 环境变量配置

项目使用 `.env` 文件管理环境变量。在项目根目录（`backend/`）下创建 `.env` 文件：

```env
# Django 配置
SECRET_KEY=django-insecure-z1)n^=!y@*r%!xt5olm+0nprawc45yt72t3m^7k5vf(x^8b43_
DEBUG=True
ALLOWED_HOSTS=*

# 数据库配置
DB_NAME=iot_monitor
DB_USER=root
DB_PASSWORD=
DB_HOST=localhost
DB_PORT=3306
MYSQL_LOCAL_INFILE=0
ENABLE_MYSQL_LOAD_DATA=0

# Redis 配置
REDIS_HOST=127.0.0.1
REDIS_PORT=6379

# Celery 配置
CELERY_BROKER_URL=redis://127.0.0.1:6379/0
CELERY_RESULT_BACKEND=redis://127.0.0.1:6379/0

# Bokeh Server 配置
BI_BOKEH_USE_SERVER=1
BI_BOKEH_SERVER_PORT=5008
BI_BOKEH_SERVER_URL=

# 缓存配置（生产环境）
DJANGO_FILE_CACHE_DIR=/path/to/cache

# 文件路径配置
PATH_CONFIG_FILE=path_config.json
WEEKLY_RESULT_FOLDER=/path/to/weekly/results
ROBOT_CONFIG_CSV=/path/to/机器人配置文件.csv
REFERENCE_DICT_CSV=/path/to/dic information.csv
ERROR_RATE_CSV_PATH=/path/to/csv/data
ERROR_RATE_CHART_PATH=/path/to/charts
```

## Django 配置参数

### 基础配置

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `SECRET_KEY` | string | *（必填）* | Django 密钥，用于加密签名 |
| `DEBUG` | boolean | `True` | 调试模式，生产环境应设为 `False` |
| `ALLOWED_HOSTS` | list | `['*']` | 允许访问的主机列表，生产环境需配置具体域名 |

### 国际化配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `LANGUAGE_CODE` | `'zh-hans'` | 语言代码（简体中文） |
| `TIME_ZONE` | `'Asia/Shanghai'` | 时区设置 |
| `USE_I18N` | `True` | 启用国际化 |
| `USE_TZ` | `True` | 启用时区支持 |

### REST Framework 配置

```python
REST_FRAMEWORK = {
    'DEFAULT_AUTHENTICATION_CLASSES': [],
    'DEFAULT_PERMISSION_CLASSES': [
        'rest_framework.permissions.AllowAny',
    ],
    'DEFAULT_PAGINATION_CLASS': 'rest_framework.pagination.PageNumberPagination',
    'PAGE_SIZE': 20,
    'DEFAULT_RENDERER_CLASSES': [
        'rest_framework.renderers.JSONRenderer',
        'rest_framework.renderers.BrowsableAPIRenderer',
    ],
}
```

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `DEFAULT_PAGINATION_CLASS` | `PageNumberPagination` | 分页类 |
| `PAGE_SIZE` | `20` | 每页数据量 |

## 数据库配置

### MySQL 连接配置

| 环境变量 | 默认值 | 说明 |
|---------|--------|------|
| `DB_NAME` | `'iot_monitor'` | 数据库名称 |
| `DB_USER` | `'root'` | 数据库用户名 |
| `DB_PASSWORD` | `''` | 数据库密码 |
| `DB_HOST` | `'localhost'` | 数据库主机地址 |
| `DB_PORT` | `'3306'` | 数据库端口 |

### 数据库选项

```python
DATABASES = {
    "default": {
        "ENGINE": "django.db.backends.mysql",
        "OPTIONS": {
            "init_command": "SET sql_mode='STRICT_TRANS_TABLES'",
            "charset": "utf8mb4",
        },
    }
}
```

### MySQL 数据导入配置

| 环境变量 | 默认值 | 说明 |
|---------|--------|------|
| `MYSQL_LOCAL_INFILE` | `'0'` | 是否启用 LOCAL INFILE（设置为 `'1'` 启用） |
| `ENABLE_MYSQL_LOAD_DATA` | `'0'` | 是否启用 LOAD DATA 导入（设置为 `'1'` 启用） |

**启用方式**：
```env
MYSQL_LOCAL_INFILE=1
ENABLE_MYSQL_LOAD_DATA=1
```

## Celery 配置

### 基础配置

| 环境变量 | 默认值 | 说明 |
|---------|--------|------|
| `CELERY_BROKER_URL` | `'redis://127.0.0.1:6379/0'` | 消息代理 URL |
| `CELERY_RESULT_BACKEND` | `'redis://127.0.0.1:6379/0'` | 结果存储后端 URL |

### 序列化配置

```python
CELERY_ACCEPT_CONTENT = ['json']
CELERY_TASK_SERIALIZER = 'json'
CELERY_RESULT_SERIALIZER = 'json'
CELERY_TIMEZONE = 'Asia/Shanghai'
```

### 定时任务配置

```python
from celery.schedules import crontab

CELERY_BEAT_SCHEDULE = {
    'auto-sync-robot-data-daily': {
        'task': 'robots.tasks.auto_sync_robot_data',
        'schedule': crontab(hour=0, minute=0),  # 每天凌晨 00:00 执行
    },
}
```

| 任务名称 | 执行时间 | 说明 |
|---------|---------|------|
| `auto-sync-robot-data-daily` | 每天 00:00 | 自动同步机器人数据 |

### Celery 任务列表

| 应用 | 任务文件 | 主要任务 |
|------|---------|----------|
| `alerts` | `alerts/tasks.py` | `check_alert_rules`, `send_alert_notifications`, `cleanup_old_alerts` |
| `robots` | `robots/tasks.py` | `auto_sync_robot_data`, `refresh_reference_dict_task`, `import_robot_components_csv_task`, `gripper_check_task` |

## Bokeh Server 配置

### 服务器配置

| 环境变量 | 默认值 | 说明 |
|---------|--------|------|
| `BI_BOKEH_USE_SERVER` | `'1'` | 是否使用 Bokeh Server（`'1'` 启用，`'0'` 禁用） |
| `BI_BOKEH_SERVER_PORT` | `'5008'` | Bokeh Server 端口号 |
| `BI_BOKEH_SERVER_URL` | `''` | Bokeh Server URL（留空自动生成） |

### 启动参数

```bash
python manage.py run_bi_bokeh_server --port 5008 --address 0.0.0.0
```

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--port` | `5008` | 监听端口 |
| `--address` | `'127.0.0.1'` | 监听地址（`0.0.0.0` 允许外部访问） |
| `--allow-websocket-origin` | 自动配置 | 允许的 WebSocket 来源 |

### 默认允许的 WebSocket 来源

- `localhost:8001`, `127.0.0.1:8001`（Django 后端）
- `localhost:5173`, `127.0.0.1:5173`（Vite 开发服务器）
- `172.20.10.3:8001`, `172.16.180.26:8001`（局域网 IP）

## CORS 配置

### 配置参数

```python
CORS_ALLOW_ALL_ORIGINS = True  # 开发环境允许所有来源
CORS_ALLOW_CREDENTIALS = True
CORS_EXPOSE_HEADERS = ["X-Server-File-Path", "Content-Disposition"]
```

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `CORS_ALLOW_ALL_ORIGINS` | `True` | 允许所有来源（生产环境建议设为 `False`） |
| `CORS_ALLOW_CREDENTIALS` | `True` | 允许携带凭证 |
| `CORS_EXPOSE_HEADERS` | 列表 | 暴露的响应头 |

### 生产环境配置

```python
CORS_ALLOW_ALL_ORIGINS = False
CORS_ALLOWED_ORIGINS = [
    "http://localhost:3000",
    "http://127.0.0.1:3000",
    "https://your-domain.com",
]
```

## 缓存配置

### 开发环境（进程内缓存）

```python
CACHES = {
    "default": {
        "BACKEND": "django.core.cache.backends.locmem.LocMemCache",
        "LOCATION": "gripper_check_cache",
        "OPTIONS": {
            "MAX_ENTRIES": 1000,
        },
    }
}
```

### 生产环境（文件缓存）

```python
CACHES = {
    "default": {
        "BACKEND": "django.core.cache.backends.filebased.FileBasedCache",
        "LOCATION": CACHE_DIR,
        "OPTIONS": {
            "MAX_ENTRIES": 1000,
        },
        "KEY_PREFIX": "sg57",
    }
}
```

| 环境变量 | 默认值 | 说明 |
|---------|--------|------|
| `DJANGO_FILE_CACHE_DIR` | `{BASE_DIR}/cache` | 文件缓存目录 |

### 缓存配置参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `MAX_ENTRIES` | `1000` | 最大缓存条目数 |
| `KEY_PREFIX` | `'sg57'` | 缓存键前缀 |

**注意**：多进程部署时请使用文件缓存或 Redis 缓存，避免使用进程内缓存。
