# RobotOps 登录界面设计风格文档

## 概述

本文档描述 RobotOps 机器人技术管理平台的登录界面设计风格。该设计采用**赛博朋克科技发布会风格**，以深邃黑色为底色，融合未来感光效与动态元素，营造高端科技产品的发布现场氛围。

---

## 整体基调

- **风格定位**: 赛博朋克 + 科技发布会
- **底色**: 纯黑 (`#000`)
- **核心感受**: 未来、科技、动态、高端

---

## 背景视觉系统

### 1. 透视网格地面

```
用途: 营造空间纵深感和前进流动感
实现: CSS perspective + rotateX 3D 变换
颜色: 青色线条 rgba(0, 204, 255, 0.15)
动画: 持续向下滚动 (20s 无限循环)
```

### 2. 蓝色主光晕

```
位置: 左上角
形态: 径向渐变椭圆光斑
颜色: rgba(0, 102, 255, 0.5) → transparent
动画: 呼吸式缩放 + 透明度变化 (4s 往复)
```

### 3. 橙金亮线

```
形态: 横贯屏幕的渐变光带
渐变: transparent → 橙金 → 青色 → transparent
动画: 左右缓慢移动 + 透明度脉冲 (6s 往复)
效果: 模仿舞台灯光扫过
```

### 4. 垂直扫描光束

```
数量: 3 条
位置: 20%, 50%, 80% 横向分布
时序: 依次延迟 0s / 2s / 4s
动画: 左右微扫 + 淡入淡出 (8s 循环)
```

### 5. 浮动粒子

```
数量: 8 个
颜色: 青色 rgba(0, 204, 255, 0.8)
轨迹: 从底部向上漂浮消散
周期: 15s 线性循环
```

## 组件样式

### 登录卡片

```css
背景: rgba(10, 10, 15, 0.85)
模糊: blur(30px)
圆角: 20px
边框: 1px solid rgba(0, 204, 255, 0.2)
阴影: 多层光晕 + 深色阴影
```

**悬停状态**:
```css
边框: rgba(0, 204, 255, 0.4)
光晕: 增强 (40px → 60px)
过渡: 0.3s ease
```

### 输入框

```css
默认: rgba(255, 255, 255, 0.03) 背景
边框: rgba(255, 255, 255, 0.1)
聚焦: rgba(0, 204, 255, 0.15) 外发光
文字: #fff
占位符: #555
```

### 按钮

```css
背景: linear-gradient(135deg, #0066ff, #00ccff)
高度: 52px
圆角: 12px
阴影: rgba(0, 102, 255, 0.3) 光晕
```

**悬停状态**:
```css
位移: translateY(-2px)
光晕: 增强扩散
```

## 设计原则

1. **层次分明**: 通过 z-index 和透明度控制，确保内容始终可读
2. **动态平衡**: 多个动画并行但不冲突，营造丰富而不杂乱的效果
3. **性能优先**: 使用 CSS 动画和 transform，避免重绘
4. **渐进增强**: 核心功能不依赖动画，动画是锦上添花
5. **视觉聚焦**: 通过光效引导视线到登录卡片
